語言技術：Servlet/JSP Gossip

Servlet 與 JSP 是一體兩面，您可以使用單獨一項技術來解決動態網頁呈現的需求，但最好的方式是取兩者的長處，JSP 是網頁設計人員導向的，而 Servlet 是程式設計人員導向的，釐清它們之間的職責可以讓兩個不同專長的團隊彼此合作，並降低相互間的牽制作用。

在前端工程當道，後端工程 Web 技術相繼攻城略地的這些年來，JSP 或許已經沒那麼重要，然而各大框架依舊支援，在框架是基於對 Servlet 的封裝而建立的情況下，瞭解 Servlet 的運作原理，仍然有其重要性。

基於 Java SE 8/Java EE 8，涵蓋 Servlet 4.0/JSP 2.3/EL 3.0。
