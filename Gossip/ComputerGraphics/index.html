非關語言: 電腦圖學入門

這邊的文件是我學習電腦圖學時的一些心得，也是入門電腦圖學的基礎，為了展現一些成果及部份的程式設計實作，使用了 Java Applet 與 HTML 5 Canvas 來展現。

這邊的資料是 2003 年初整理居多，2021 年以後，我重新整理了一系列圖學相關資料，放在〈玩轉 p5.js〉，跟 3D 建模應用相關的，則是整理在〈OpenSCAD〉，有興趣可以進一步參考。
